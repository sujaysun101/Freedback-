# FeedbackFix - Client Feedback Translator

Transform vague client feedback into actionable design tasks. Save time, reduce revision cycles, and improve client communication.

## Product Overview

FeedbackFix translates messy client feedback ("make it pop", "needs more pizzazz") into specific, actionable design tasks ("increase contrast on the headline by 40%", "add 20px padding around the CTA button").

## Tech Stack

- **Frontend**: Next.js 14 (React)
- **Backend**: Python FastAPI
- **Database**: PostgreSQL
- **AI**: OpenAI GPT-4/3.5-Turbo
- **Authentication**: Supabase Auth
- **Payments**: Stripe
- **Deployment**: Vercel (Frontend), Railway/Render (Backend)
